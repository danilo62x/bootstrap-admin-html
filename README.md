# Bootstrap Admin / HTML

> Admin estático em HTML + Bootstrap 5.3

[![Licença: MIT](https://img.shields.io/badge/Licen%C3%A7a-MIT-blue.svg)](./LICENSE) ![Grátis](https://img.shields.io/badge/pre%C3%A7o-Gr%C3%A1tis-brightgreen)

Painel administrativo estático em HTML + JS vanilla + Bootstrap 5.3 (SCSS compilado). 25 páginas, i18n (pt-BR/en/es), PWA, ApexCharts, Quill e flatpickr via CDN. Sem build de framework — só abrir.

## 🧱 Stack

- HTML5
- JavaScript
- Bootstrap 5.3
- Sass

## ✨ Recursos

- HTML + CSS + JS vanilla — zero framework
- Bootstrap 5.3 (SCSS compilado via sass)
- 25 páginas estáticas idênticas às demais famílias
- i18n pt-BR/en/es, PWA (service worker)
- ApexCharts, Quill e flatpickr via CDN
- Estado/auth/tema em JS vanilla, dark mode
- Layout responsivo, toasts e modais

## 🚀 Como rodar

As páginas são estáticas — abra o `index.html` no navegador. Para recompilar o SCSS ou servir com live reload:

```bash
npm install
npm run dev
```

Para regenerar o CSS de produção: `npm run build`.
